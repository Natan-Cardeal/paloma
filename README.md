# Paloma

Plataforma de Relatorios de Sustentabilidade para PMEs brasileiras.

**Produtos**: PGRS | Inventario de Carbono GEE | Relatorio de Sustentabilidade

## Stack

- Next.js 15 (App Router) + TypeScript
- MDX (content in repo)
- Tailwind CSS
- Vercel (hosting + ISR + OG images)
